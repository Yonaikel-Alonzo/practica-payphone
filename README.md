# Practica PayPhone - Jenkins CI/CD

Proyecto web desplegado automáticamente con Jenkins y Docker usando npx live-server.

## Tecnologías
- Jenkins
- Docker
- Node.js
- live-server

## Flujo CI/CD
Cada push en main ejecuta el pipeline que construye la imagen y despliega el contenedor automáticamente.

## Puerto
http://localhost:3001
